# Next Sense — BIC Design System

### 🔗 [**Open the live site →**](https://maudcode.github.io/next-sense-bic-design-system/)

> https://maudcode.github.io/next-sense-bic-design-system/

The documentation site for the **BIC (Building Intelligence Center)** design system — colors, typography, spacing, components and patterns, grounded in the real product code.

Built with Vite + React + TypeScript + Tailwind v4 + shadcn/ui.

---

## Pages

- **Foundations** — [Colors](https://maudcode.github.io/next-sense-bic-design-system/colors) · [Typography](https://maudcode.github.io/next-sense-bic-design-system/typography) · [Spacing](https://maudcode.github.io/next-sense-bic-design-system/spacing) · [Radii](https://maudcode.github.io/next-sense-bic-design-system/radii) · [Shadows](https://maudcode.github.io/next-sense-bic-design-system/shadows) · [Iconography](https://maudcode.github.io/next-sense-bic-design-system/iconography)
- **Components** — Button · Badge · Input · Card · Table · Tabs · Segmented Control · Charts
- **Patterns** — Filter Bar · Widgets

## Run locally

```bash
npm install
npm run dev        # http://localhost:5173
```

## Deploy

Pushing to `main` auto-deploys to GitHub Pages via the workflow in `.github/workflows/deploy.yml`.
